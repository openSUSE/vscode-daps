= Introduction

== Features

The DAPS tool helps you author and publish documentation written in DocBook XML and AsciiDoc.
This extension makes it easier to run selected DAPS commands from the VSCode editor environment.

== Requirements

This extension requires that you...

* install DAPS on your system. Refer to https://opensuse.github.io/daps/ for
  more details.
