1 Introduction #

 

1.1 Features #

 

The DAPS tool helps you author and publish documentation written in DocBook XML and AsciiDoc. This extension makes it easier to run selected DAPS commands from the VSCode editor environment.

1.2 Requirements #

 

This extension requires that you…​

  • install DAPS on your system. Refer to https://opensuse.github.io/daps/ for more details.

  • install the AsciiDoc extension to VSCode

  • install this DAPSy` extension to VScode

1.3 Known Issues #

 

See our issue tracker at openSUSE/vscode-daps/issues.
